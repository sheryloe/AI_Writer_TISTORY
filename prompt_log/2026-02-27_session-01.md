## 노션 현재 상태
- `NOTION_PARENT_PAGE_ID`는 설정되어 있으나, 해당 페이지가 Integration과 공유되지 않아 `object_not_found` 오류 발생
- 조치 필요: 대상 페이지 우상단 `...` -> `Connections`에서 현재 Integration 연결(공유) 후 재시도
- Step 1/2 본문 템플릿을 UTF-8 한국어 내용으로 재작성 완료
- 재확인: `NOTION_PARENT_PAGE_ID=31461b8e...deee` 값 로딩은 정상이나, API 접근은 동일하게 `object_not_found`
- 추가 진단: `users.me` 호출은 성공(`AUTH_OK user bot`), `search` 결과 페이지 0건으로 통합에 공유된 페이지가 없는 상태 확인
