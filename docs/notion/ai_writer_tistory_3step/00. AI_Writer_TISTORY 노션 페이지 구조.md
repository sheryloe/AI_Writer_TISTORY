# AI_Writer_TISTORY 노션 페이지 구조 (3-Step)

상위 페이지 제목은 **AI_Writer_TISTORY | 티스토리 글쓰기 자동화 서버 제작기** 로 두고, 아래 3개 하위 페이지를 넣으면 됩니다.

## 권장 설명

티스토리 글 초안 생성, 발행 파이프라인, 운영 로그를 백엔드 중심으로 정리한 3단계 노션 원고 세트입니다.

## 권장 페이지 트리

- Step 1. 티스토리 자동화 서버를 별도 레포로 분리한 이유
- Step 2. Prompt log와 SQLite를 붙여 백엔드를 운영형으로 만든 과정
- Step 3. 티스토리 발행 파이프라인과 Notion 운영 로그를 묶은 결과
